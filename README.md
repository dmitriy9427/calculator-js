# Калькулятор

- CSS3
- JS

## Функциональность

- Сложение
- Вычитание

### Инструкция по установке:

**Склонируйте репозиторий к себе на компьютер**

```
git clone https://github.com/dmitriy9427/calculator-js.git
```

**Запустите у себя на редакторе кода с помощью Live Server**
